Это десктопное приложение на Windows Forms (.NET 8+) для учета компьютерной техники на предприятии. Система позволяет вести справочники (подразделения, сотрудники, типы оборудования), управлять оборудованием (CRUD, история перемещений), учитывать лицензионное ПО (многие-ко-многим) и генерировать простые отчеты.
Цель: Закрепление навыков разработки, работы с БД, тестирования и документации в едином проекте.
Функционал:

Справочники: Подразделения, Сотрудники, Типы оборудования (CRUD).
Учет оборудования: Оборудование (CRUD, статус, ответственный), История перемещений.
Учет ПО: Лицензии (CRUD), Установленное ПО (связь с оборудованием, дата установки).
Отчеты: Оборудование по подразделениям, ПО на компьютере сотрудника.
Интерфейс: MDI с меню, TreeView-навигацией, статус-баром, валидацией.
